Exploratory Data Analysis (EDA) on Retail Sales Data
Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on a retail sales dataset to identify patterns, trends, and insights that can help businesses make informed decisions. The analysis involves data cleaning, statistical analysis, and visualization to better understand customer behavior and product performance.

This project was completed as part of the Oasis Infobyte Internship Program.

Objectives

Load and clean the retail sales dataset.

Perform descriptive statistical analysis.

Analyze sales trends over time.

Understand customer demographics and purchasing behavior.

Visualize insights using charts and graphs.

Provide actionable recommendations based on the analysis.

Dataset Information

The dataset contains retail transaction details including:

Transaction ID – Unique identifier for each transaction

Date – Date of purchase

Customer ID – Unique identifier for customers

Gender – Customer gender

Age – Customer age

Product Category – Category of the purchased product

Quantity – Number of items purchased

Price per Unit – Price of each item

Total Amount – Total transaction amount

Steps Performed in the Project
1. Data Loading

The dataset was loaded using the Pandas library.

2. Data Cleaning

Checked for missing values

Converted date column to datetime format

Ensured correct data types

3. Descriptive Statistics

Basic statistics such as mean, median, standard deviation, minimum, and maximum values were calculated to understand the dataset.

4. Time Series Analysis

Sales trends were analyzed over time to identify patterns and fluctuations.

5. Customer Analysis

Customer demographics such as age and gender were analyzed to understand purchasing behavior.

6. Product Category Analysis

Total sales were analyzed by product category to identify the most popular products.

7. Data Visualization

Various visualizations were created including:

Line plots for sales trends

Bar charts for product category sales

Histograms for age distribution

Heatmaps for correlation analysis

Scatter plots for quantity vs sales

Key Insights

Certain product categories generated higher sales compared to others.

Sales trends varied across different dates indicating potential seasonal patterns.

Customer demographics such as age and gender influenced purchasing behavior.

Higher quantities purchased generally resulted in higher total sales.

Recommendations

Focus marketing strategies on high-performing product categories.

Target promotions toward active customer demographics.

Offer discounts on bulk purchases to increase sales volume.

Monitor seasonal sales trends to improve inventory planning.
